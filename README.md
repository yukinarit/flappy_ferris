Flappy Ferris
=============

![](demo.gif)

Setup
-----

* Debian / Ubuntu
    ```
	$ sudo apt install libudev-dev zlib1g-dev alsa libasound2-dev
	```
* CentOS / RHEL
    ```bash
	$ sudo yum install systemd-devel zlib-devel alsa-lib-devel
	```

Run
---

* Run on macOS/Linux
    ```bash
    $ cargo run
    ```

* Run as WebAssembly
    ```bash
    $ cargo web start
    ```
